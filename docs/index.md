# LeetCode Solutions

??? info "How do the filters work?"

    - **Rating** uses zerotrac ratings. Click on the arrow to sort by ascending rating,
    descending, or neutral (sort by default problem id number)
    - **Topics** allow filtering by problems that have a specific topic. for multiselect, a problem must contain every selected topic to be displayed.
    - **Hide unrated** excludes problems without a zerotrac rating. Since zerotrac tracks
    contests, some problems have a hyphen to signal they are unrated and by default appear in every filter.
